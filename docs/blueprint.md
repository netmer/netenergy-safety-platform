# **App Name**: NetEnergy Safety Platform

## Core Features:

- Advanced Course Filtering: Intuitive course search with filtering by category, role, and month.
- Comprehensive Course Details: Detailed course pages with tabs for overview, schedule, in-house requests, content, and FAQs.
- Easy Registration & Payment: Streamlined registration and payment flow with automated invoice generation.
- Centralized Admin Panel: Admin panel for managing courses, schedules, registrations, and client data.
- AI Compliance Tool: Compliance Checker tool analyzes user input to recommend legally required training courses; will use a reasoning tool to interpret different user inputs to determine if a specific piece of information should be included in the results.
- Client Portal: Personalized dashboard for organizational clients to track training history and manage employee data.
- Blog: Blog section to provide educational artcles related to health and safety training

## Style Guidelines:

- Primary color: Navy Blue (#00308F) for a professional and trustworthy feel, based on the original brand color.
- Background color: Light blue (#E0F2F7), a desaturated version of the primary, creating a calm backdrop.
- Accent color: Orange (#FF8C00) for call-to-action buttons to draw attention.
- Body and headline font: 'Kanit' (sans-serif) for clear readability on all devices.
- Mobile-first and responsive design to ensure optimal viewing experience on all devices.
- Intuitive navigation with a clear menu structure, ensuring users can find information within 3 clicks.
- Subtle transitions and loading animations to enhance user experience without being intrusive.
